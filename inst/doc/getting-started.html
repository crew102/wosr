Getting started

  1. The first step is to open up a session with the WoS API. To do this, you call auth(), which authenticates the user with the API’s server and returns a session ID (SID).
library(wosr)
sid <- auth(username = "your_username", password = "your_password")
  1. Now we can query WoS to see how many records match our query string.
# Find all publications that contain "animal welfare" in their titles (TI tag)
# and have the words "dog" and "welfare" somewhere in their titles, abstracts, or
# list of keywords (TS tag).
query <- 'TI = ("animal welfare") AND TS = (dog welfare)'
query_wos(query, sid = sid)
#> Matching records: 82
  1. Pull the data.
data <- pull_wos(query, sid = sid)

  1. pull_wos() returns a series of data frames that are like tables in a relational database. You an link these data frames together as needed, to answer whatever questions you have. For example:
  • What are the 5 most popular journal subject categories (JSCs) in this set of publications and which publications belong to them?
library(dplyr)

top_jscs <- 
  data$jsc %>% 
    group_by(jsc) %>% 
    count() %>% 
    arrange(desc(n)) %>% 
    head()

  • Which publications have “cat” in their abstracts, who are the authors on these publications, and which organizations are they from?
cat_pubs <- 
  data$publication %>% 
    filter(grepl("\\bcat\\b", abstract, ignore.case = TRUE)) %>% 
    select(ut, title)

  1. Download more detailed citation data (from the InCites API) for the top-cited publications
top_100_pubs <- 
  data$publication %>% 
    arrange(desc(tot_cites)) %>% 
    slice(1:100) %>% 
    .$ut
